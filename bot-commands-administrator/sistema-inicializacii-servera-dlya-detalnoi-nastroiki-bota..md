# Система инициализации сервера для детальной настройки бота.

## !init

Начнет инициализацию сервера в базу данных.

```text
!init
```

## !setdefaultrole \(id роли\)

Установит роль с \(id роли\) как роль по умолчанию \(авто-роль\).
